## Placeholder Fields

| Field | Description |
|---|---|
| `[YEAR]` | The year of initial publication. Use a range for multi-year projects (e.g., `2025-2026`). |
| `[COPYRIGHT_HOLDER]` | The legal entity or individual holding copyright. For Core Red projects, use `Sxnnyside Project`. |

## Rules

- The legal body of the license **must not** be altered, reformatted, or paraphrased.
- Placeholder fields **must** be replaced with valid values before commit.
- The completed license **must** be placed in a file named `LICENSE` at the repository root.
- No additional license text, preamble, or commentary **may** be added to the `LICENSE` file.
